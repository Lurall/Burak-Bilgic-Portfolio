# Burak Bilgiç | Digital Portfolio & CV

Software Development Engineer in Test (SDET) & Geliştirici portfolyosu. Bu proje, standart bir özgeçmişin ötesine geçerek kullanıcıya interaktif, işitsel ve görsel bir deneyim sunmak amacıyla geliştirilmiştir.

## 🚀 Özellikler (Features)
- Dinamik Scroll Deneyimi: Sayfa kaydırıldıkça gün doğumundan gün batımına geçen özel tasarım.
- İşitsel Etkileşim: Form alanlarında "Creamy Switch" mekanik klavye sesleri ve entegre Lo-Fi müzik çalar.
- Görsel Efektler: Framer Motion ile kodlanmış "Zıplayan Kar" parçacıkları ve solid/gradyansız modern UI.
- Aktif İletişim Formu: Doğrudan e-posta yönlendirmeli, bildirimi olan iletişim altyapısı.

## 💻 Teknoloji Yığını (Tech Stack)
- Frontend: Next.js (App Router), React
- Stil ve Animasyon: Tailwind CSS, Framer Motion
- Form Yönetimi: EmailJS (veya kullanılan servis)
- Ses Yönetimi: HTML5 Audio API

## 🛠️ Kurulum (Local Development)
Projeyi kendi bilgisayarınızda çalıştırmak için:
1. `npm install`
2. `npm run dev`
